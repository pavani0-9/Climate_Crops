# Climate Crop Intelligence Dashboard

A **Flask-based web application** to visualize climate projections, crop yield forecasts, and KPIs for different regions. The dashboard allows users to select country, state, crop, and years to see interactive climate and yield data.

## Live Demo
URL: https://climate-crops.onrender.com


## Features
- **Location & Crop Selection:** Choose country, state, crop, start year, and end year.  
- **Climate Projections:** View temperature (°C), rainfall (mm), and CO2 (ppm) trends.  
- **Yield Forecast:** See predicted crop yield indices over time.  
- **KPIs:**  
  - Yield Change (%)  
  - Water Stress (%)  
  - Pest Risk (%)  
  - Food Security (%)  
- **Interactive Charts:** Built with **Chart.js** for dynamic visualization.

## Technologies Used
- **Backend:** Flask, Python  
- **Data & Analysis:** pandas, numpy, scikit-learn, joblib  
- **Frontend:** HTML, CSS, JavaScript, Chart.js  
- **Deployment:** Render
